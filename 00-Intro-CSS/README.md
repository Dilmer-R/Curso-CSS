# Tutorial CSS

CSS es un lenguaje que se usa para dar estilos a un documento HTML.

CSS significa, Cascading Style Sheet. Hojas de estilo en cascada.

Ejemplo:

``` css
h1 {
    color: red;
}

h1 {
    color: yellow;
}
```
