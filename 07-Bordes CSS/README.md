#  Bordes en CSS

Sirven para especificar el estilo, el ancho y el color del borde de un elemento.

* Propiedad: 

```css
                dotted;
                dashed;
                solid; 
                double;
                groove;
border-style:
                ridge;
                inset;
                outset;
                none;
                hidden;
```

1. Borde dotted:
```css
border-style: dotted;
```
2. Borde dashed:
```css
border-style: dashed;
```
3. Borde solid:
```css
border-style: solid;
```
4. Borde double:
```css
border-style: double;
```
5. Borde groove:
```css
border-style: groove;
```
6. Borde ridge:
```css
border-style: ridge;
```
7. Borde inset:
```css
border-style: inset;
```
8. Borde outset:
```css
border-style: outset;
```
9. Borde none:
```css
border-style: none;
```
10. Borde hidden:
```css
border-style: hidden;
```
11. Borde mixed:
```css
border-style: dotted dashed solid double;
```

- Ver todo esto en la ```demo```.
